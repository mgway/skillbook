Adding an API Key

What permissions are required?
Skillbook only needs CharacterInfo (private), SkillQueue, and CharacterSheet permissions. You can supply more, but the key will be rejected if don't supply those three. Follow this link create a key that has the correct permissions automatically filled in.

Why can't I add this key?
Did it have the required permissions? Check the previous answer for more information.
Have you added this key to your account already? While other accounts can have the same key without issue, you can't add the same key to your account twice. For example, it is okay if you gave a key to a corporation's recruiter who used it on Skillbook and then try to use the same key yourself.

Character sheet

I added something to my queue but it isn't showing up on skillbook
Be patient. The EVE API caches the skill queue for a length of time (variable, usually 15-30 minutes), so it may take a while for the queue to update. If it hasn't updated in more than an hour, try refreshing the page. If the problem persists, consider reporting a bug (see below).
